# My Tea Reviews

A modern, sleek, and responsive tea review website hosted on Cloudflare Pages.

## About
Hosted at [myteareviews.com](https://myteareviews.com). This site provides a personal ranking of teas, along with "Teasonality" quiz.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Run locally**:
    ```bash
    npm run dev
    ```
